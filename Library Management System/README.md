# Library Book Management System 📚
### FastAPI Internship Final Project - Innomatics Research Labs

This is a fully functional backend system built with **FastAPI** as part of the 6-day intensive internship. The project demonstrates core backend development skills, including REST API design, Pydantic validation, CRUD operations, and advanced data handling.

## 🚀 Features
- **Total Endpoints:** 20 (Covering Beginner to Advanced tasks).
- **CRUD Operations:** Full lifecycle management for Books and Members.
- **Data Validation:** Strict type checking and logic using Pydantic schemas.
- **Advanced Logic:** Implementation of Keyword Search, Multi-field Sorting, and Pagination.
- **Workflow:** Integrated "Borrowing" system that links members and books.

## 🛠️ Tech Stack
- **Framework:** FastAPI
- **Web Server:** Uvicorn
- **Language:** Python 3.x
- **Validation:** Pydantic v2
